### 1. 查看环境变量 <span id = "查看环境变量">

查看环境变量有三个命令：

- `env`：env命令是environment的缩写，用于列出所有的环境变量。
- `export`：单独使用export命令也可以像env列出所有的环境变量，不过export命令还有其他额外的功能。
- `echo $PATH`： echo $PATH用于列出变量PATH的值，里面包含了已添加的目录。
